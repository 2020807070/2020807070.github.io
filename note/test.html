基于VuePress生成的一款博客系统。

# 预览地址(http://123hao123.top)

# 目录

  • 相关依赖
  • 开发指南
  • 项目目录结构

# 相关依赖

  1. VuePress：Vue 驱动的静态网站生成器;
  2. Valine：无后端评论系统;

# 开发指南

# 开发环境，启动服务

$ npm run start

# 开发环境，调试显示console

$ npm run dev

# 生产环境，构建项目

$ npm run build

# 部署(TODO: 待完善)

$ npm run upload

# 项目目录结构

|-- docs
| |-- .vuepress             项目的开发目录
| | |-- components          自定义组件
| | |-- nav                 右上方导航
| | | |-- index.js
| | |-- public              静态资源
| | | |-- logo.png          网站logo
| | |-- theme               自定义主题
| | | |-- config.js         配置文件
| | |-- index等...          页面目录
| | |-- README.md           自定义首页
|-- dist                    存储构建之后的项目目录
|-- node_modules
|-- .gitignore
|-- package.json            package配置文件
|-- package-lock.json
|-- README.md               你当前所看到的项目说明
